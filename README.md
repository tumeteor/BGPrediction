# BGPrediction

This documentation is for the Glycorec Blood Glucose Prediction. 

### Prerequisites

Python (>= 2.7 or >= 3.3,  
NumPy (>= 1.8.2),  
SciPy (>= 0.13.3),  
Scikit-learn (>=0.16),  
APScheduler (>=3.3.0),  
mysql-connector-python (>=2.0),  
Pandas (>=0.12.0).

## Getting Started

Add the project folder to the Python path:

```
export PYTHONPATH=$PYTHONPATH:/path/to/the/project
```
